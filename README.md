# Relational Commits

A sensical, opinionated commit format.

[Specification](./spec/index.md)

## Use

```bash
$ pnpm add @relagit/commits -g

$ relational
Usage: relational <command> [args]

Commands:
  parse <message>  Parse a commit message
  map <path>       Generate a map of commit history
```
